## Current Voice AI Agents (Built with Vapi)
All created by prompting + Grok/Claude for plans/docs, then Vapi for voice setup. Learning custom tools, phone # assignment, file uploads, knowledge PDFs, MMS one-pagers, API key extraction, Twilio/Make.com flows, Serper search pulls.

- **Empath HR Intake Agent** 🏢  
  **Purpose:** HR call-in agent that collects employee permit/case data conversationally. Converts full intake into clean PDF → auto-sends to caseworkers.  
  **Impact:** Saves HR massive time/headaches – no more messy forms or follow-ups.  
  **Built:** Last night (~10 PM PDT). Used Grok for overall plan, Claude to generate structured docs/PDF templates.  
  **Status:** Live / Testing calls.  
  **Key Learnings:** Knowledge base PDFs → compress → upload to Vapi; tool for PDF generation.
